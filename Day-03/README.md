## 📚 Commands Learned

### 1. whoami

Purpose:
Displays the currently logged-in user.

Example:

```bash
whoami
```

### 3. groups

Purpose:
Shows all groups the current user belongs to.

Example:

```bash

```

### 4. who

Purpose:
Displays users currently logged into the system.

Example:

```bash
who
```

---


### 5. sudo

Purpose:
Runs commands with administrator privileges.

Example:

```bash
sudo apt update
```

---

## 📝 What I Learned Today

- Linux identifies every user using a User ID (UID).
- Files have an owner and a group.
- Groups make permission management easier.
- `sudo` allows temporary administrator access.
